docker의

Proxy로 사용할 녀석 하나(hub) / Storage로 사용할 녀석 하나(hosted)

이렇게 총 2개를 만들어줍니다.

Docker Hub Proxy 생성

Name: docker-hub
Repository Connectors에서 HTTP: 8082 포트로 설정
Docker Index: Use Docker Hub

Docker Hosted 생성

Name: docker-hosted
Repository Connectors에서 HTTP: 8083 포트로 설정
Allow anonymous docker pull: 체크
