Kubernetes 마스터 플랜 (CKA 취득 목표)

작성일 2025-09-17 Edited on 2025-09-19 In Kubernetes , DevOps , Learning

목표: 8주 만에 CKA 취득

최종 목표:

  • CKA (Certified Kubernetes Administrator) 취득
  • 실무에서 바로 쓸 수 있는 K8s 운영 능력
  • 클라우드 네이티브 아키텍처 설계 역량

학습 방식: 이론 30% + 실습 70%

8주 학습 계획

Week 1-2: 기초 다지기

환경 구축

핵심 리소스

  • Pod, ReplicaSet, Deployment
  • Service (ClusterIP, NodePort, LoadBalancer)
  • Ingress, ConfigMap, Secret, Namespace

실습 예제

Week 3-4: 고급 기능

스토리지, 네트워킹, 보안

  • PersistentVolume, PersistentVolumeClaim
  • StorageClass (동적 프로비저닝)
  • NetworkPolicy, RBAC, SecurityContext

RBAC 예제

Week 5-6: 클러스터 운영

핵심 스킬

  • 클러스터 업그레이드 (kubeadm)
  • etcd 백업 및 복구
  • 로깅 및 모니터링
  • 트러블슈팅

etcd 백업 (CKA 시험 필수!)

Week 7-8: CKA 시험 준비

시험 도메인

  1. 클러스터 아키텍처, 설치 및 구성 (25%)
  2. 워크로드 및 스케줄링 (15%)
  3. 서비스 및 네트워킹 (20%)
  4. 스토리지 (10%)
  5. 트러블슈팅 (30%)

학습 리소스

필수 (무료)

  • Kubernetes 공식 문서
  • kubectl Cheat Sheet
  • GitHub 실습 레포

유료 (추천)

  • Udemy - Mumshad Mannambeth “CKA with Practice Tests”
  • A Cloud Guru “Certified Kubernetes Administrator”

실습 환경

주간 체크리스트

Week 1

  • kubectl 기본 명령어 50개 암기
  • Pod 생성/삭제/수정 1분 내 완료
  • Service로 Pod 노출
  • ConfigMap/Secret 활용

Week 2

  • Deployment 롤링 업데이트
  • Ingress 외부 트래픽 라우팅
  • Namespace 멀티 테넌트 구성

Week 3-4

  • PV/PVC 동적 프로비저닝
  • RBAC 세밀한 권한 관리
  • NetworkPolicy 마이크로세그멘테이션

Week 5-6

  • etcd 백업/복구 10회 이상 연습
  • 클러스터 업그레이드 시나리오
  • 장애 트러블슈팅 20개 케이스

Week 7-8

  • Killer.sh 모의시험 5회 이상 (80점 이상)
  • 시험 환경 실전 연습
  • CKA 시험 접수 및 응시

실무 적용

사내 프로젝트 제안

1단계: 기존 서비스 컨테이너화

2단계: K8s 매니페스트

3단계: CI/CD 파이프라인

시험 합격 팁

  1. 시간 관리: 문제당 평균 7분
  2. kubectl 숙련도: 타이핑 속도 향상
  3. 공식 문서 활용: 시험 중 참고 가능
  4. 실전 연습: Killer.sh 반복

CKA 취득 후 로드맵

  • CKAD (Developer) 취득
  • CKS (Security) 취득
  • 클라우드별 전문 자격증 (AWS EKS, GCP GKE)
  • 기술 리더십 역할

전체 학습 과정에서의 위치

이 Kubernetes 학습은 전체 성장 로드맵의 5단계에 해당한다.

이전 단계들:

  1. AI 시대 현황 분석
  2. Git 저장소 관리 (작성 중)
  3. Redis Docker 환경 구축 (작성 중)
  4. Jenkins CI/CD 파이프라인 (작성 중)

다음 단계들:
6. Prometheus & Grafana 모니터링 (작성 예정)
7. AWS 클라우드 아키텍처 (작성 예정)
